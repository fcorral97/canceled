		on your machine.
		Please see the following documentation:
			https://docs.docker.com/get-docker/
	
	PYTHON: This web application requires the latest version of Python to be installed
		Please see the following documentation:
			https://www.python.org/downloads/ 
		
*****INSTRUCTIONS ON HOW TO START RUNNING THE CANCELED APPLICATION***************

1. Open either terminal or Command prompt depending on your machine.
   Change directory to the "Canceled" directory. (This should be where 
   'docker-compose.yml' file is located)

2. Run the following command: docker-compose up -d

3. Once the containers are up and running. You should be able to open your browser and
   Navigate to: "http://0.0.0.0"

4. There you will be able to interact with application

5. To bring down the containers. Run the following command: docker-compose down
